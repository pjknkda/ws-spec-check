# ws-spec-check
Websocket specification checker based on autobahn-testsuite.

Target libraries:

- aiohttp (2.1.0)
- aiohttp (2.1.0) + uvloop (0.8.0)
- tornado (4.5.1)
- websockets (3.3)

You can see the report [here](https://rawgit.com/pjknkda/ws-spec-check/master/reports/fuzzingclient/index.html).
